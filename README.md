# RoastGit

AI-powered GitHub profile roasts. Enter any username, get an honest review.

**Live:** [roastgit.in](https://roastgit.in)

## Stack

- Next.js 14 (App Router)
- TypeScript
- Redis (caching)
- GitHub REST API
- Tailwind CSS

## Local Development

```bash
npm install
npm run dev
```

Copy `.env.example` to `.env` and fill in your keys.
